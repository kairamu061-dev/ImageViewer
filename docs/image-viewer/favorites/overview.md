# favorites 概要

## 目的・背景

よく使うフォルダを登録・管理するお気に入り機能。仮想グループで整理でき、D&D で並び替えができる。通常タブとは独立した専用タブとして提供する。

## スコープ

### 作るもの

- お気に入りタブ（同時に1つのみ）
- フォルダ登録（通常タブのコンテキストメニューから）
- 仮想グループ（「階層」）の作成・名前変更・削除
- D&D による並び替えとグループへの移動
- お気に入りパネルの `<`/`>` 折りたたみボタン

### 作らないもの

- フォルダへのメモ・タグ付け
- お気に入いのインポート/エクスポート
- 実フォルダの作成・削除

## 制約

- お気に入いは同時に1タブのみ（複数開けない）
- 仮想グループはアプリ内管理のみで実フォルダとは無関係

## 完了条件

- フォルダをお気に入りに追加でき、一覧に表示される
- 階層グループを作成・名前変更・削除できる
- D&D でフォルダ・グループを自由に並び替えできる
- お気に入りのフォルダをクリックすると右パネルに画像が表示される
- 状態が保存され次回起動時に復元される
