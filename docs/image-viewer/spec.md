# image-viewer 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | フォルダ選択 | ダイアログでルートフォルダを選択してツリーを構築 |
| 2 | フォルダツリー表示 | サブフォルダのみを階層表示（折りたたみ/展開） |
| 3 | 画像フィット表示 | ビューアーサイズに合わせて画像を拡縮表示 |
| 4 | ホイールズーム | マウスホイールで画像をズームイン・アウト |
| 5 | サイドボタン切り替え | マウスの進む/戻るボタンで画像を前後に切り替え（循環） |
| 6 | ホバースライダー | ビューアー下部ホバーでスライダーを表示、画像切り替え |
| 7 | 画像プリロード | 隣接画像をバックグラウンド先読みして高速切り替え |

## 画面・操作フロー

1. アプリ起動 → フォルダ選択ダイアログが表示される
2. フォルダを選択 → 左パネルにサブフォルダツリーが表示される
3. ツリーのフォルダをクリック → 右パネルに先頭画像が表示される
4. マウスサイドボタン or スライダーで画像を切り替える
5. マウスホイールで画像のズームを調整する

## 各画面 / 状態の詳細

### フォルダツリーパネル（左）

- **表示内容**: ルートフォルダ名をトップに、サブフォルダのみをツリー形式で表示。ファイルは表示しない。
- **操作**:
  - フォルダ行をクリック → そのフォルダを選択し画像ビューアーを更新
  - ▶/▼アイコンクリック → フォルダの展開/折りたたみ
  - ツールバーの「フォルダを開く」→ フォルダ選択ダイアログを再表示
- **選択状態**: 現在選択中のフォルダをハイライト表示

### 画像ビューアーパネル（右）

- **表示内容**: 選択フォルダの現在インデックスの画像。デフォルトはフィット表示（縦横比を保ちつつビューアー全体に収める）
- **操作**:
  - マウスホイール上 → ズームイン（+10% ステップ）
  - マウスホイール下 → ズームアウト（-10% ステップ、最小 10%）
  - マウスサイドボタン「進む」(XButton2) → 次の画像（末尾から先頭に循環）
  - マウスサイドボタン「戻る」(XButton1) → 前の画像（先頭から末尾に循環）
  - ビューアー下部20pxへのホバー → スライダーを表示
  - スライダー外へのホバー離脱 → 500ms 後にスライダーを非表示

### ホバースライダー

- **表示条件**: 選択フォルダに2枚以上の画像がある場合のみ表示
- **表示タイミング**: ビューアー下部 20px 領域へのマウス進入時
- **レイアウト**: ビューアー下部に半透明のオーバーレイとして表示
- **動作**:
  - スライダー右端 = 先頭画像（インデックス 0）
  - スライダー左端 = 末尾画像（インデックス N-1）
  - ドラッグまたはクリックで任意の画像にジャンプ

### 画像なし状態

- **表示内容**: 選択フォルダに対応画像がない場合、「画像がありません」メッセージを中央に表示
- **スライダー**: 非表示

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 選択フォルダに読み込める画像がない | ビューアーに「画像がありません」を表示 |
| 画像ファイルが破損・読み込み失敗 | その画像をスキップしてリストから除外 |
| フォルダ選択ダイアログでキャンセル | 現在の状態を維持（初回起動時はそのまま空白表示） |

## 未対応ケース

- GIF アニメーション（静止画として先頭フレームのみ表示）
- RAW 形式（CR2, NEF 等）
- ネットワークドライブ上の大量画像フォルダでのパフォーマンス保証
- マルチモニター環境でのウィンドウ位置記憶
