# tabs 開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| setTabsClosable を使わず setTabButton でカスタム × ボタン | デフォルトのクローズアイコンがダーク背景で不可視だったため手動実装 |
| タブタイトル更新を title_changed シグナルで行う | TabContent が自分のタイトルを知っており、indexOf で逆引きする方が疎結合 |
| スワップモードをグローバル管理（_default_swap_mode）| タブごとに異なる設定は UX として複雑すぎるため |

## 発生した問題と対処

| 問題 | 対処 |
|------|------|
| デフォルトの × アイコンが暗い背景で見えない | QToolButton("×") を手動で setTabButton に配置 |
| タブが背景と同化して見分けにくい | 選択タブに #3c3c3c 背景 + 上端 3px 青ボーダーを適用 |
