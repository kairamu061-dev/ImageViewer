# tabs 概要

## 目的・背景

複数のフォルダを同時に開き、タブで切り替えながら閲覧できるようにする。各タブはフォルダビューと画像ビューアーを独立して持ち、タブタイトルで現在の状態を確認できる。

## スコープ

### 作るもの

- QTabWidget ベースのマルチタブ UI
- タブタイトル（「フォルダ名 | 画像名」形式）
- `+` ボタンで新規タブ追加
- `×` ボタンでタブを閉じる（最後の1枚は閉じない）
- `☆/★` ボタンでお気に入りタブを開閉
- タブを閉じても最低1枚の通常タブを維持

### 作らないもの

- タブのドラッグで別ウィンドウに切り出し
- タブのキーボードショートカット（Ctrl+T, Ctrl+W など）

## 制約

- 各タブが独立した FolderTreePanel + ImageViewerPanel を持つ
- スワップモードはタブ横断のグローバル設定

## 完了条件

- 複数タブを開いてそれぞれ独立したフォルダ・画像を表示できる
- タブタイトルが選択フォルダ・表示画像名に連動して更新される
- 閉じるボタンが常時表示されクリックで閉じられる
- 最後の通常タブは閉じられない
