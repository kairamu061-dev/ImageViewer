# tabs 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | マルチタブ | 複数タブを同時に開き独立して操作できる |
| 2 | タブタイトル | 「フォルダ名 \| 画像名」形式。画像のない場合はフォルダ名のみ |
| 3 | 新規タブ | `+` ボタンで空のタブを追加。コンテキストメニューの「別のタブとして開く」でもフォルダ付きで追加 |
| 4 | タブを閉じる | 各タブ右端の `×` ボタン。最後の通常タブは閉じられない |
| 5 | お気に入りタブ | `☆` ボタンで開く（同時に1つだけ）。`★` に変わり閉じると `☆` に戻る |

## 画面・操作フロー

1. 起動時 → 前回の状態を復元（初回は空のタブ1枚）
2. `+` クリック → 新しいタブ（タイトル「新しいタブ」）が追加され選択される
3. フォルダを開く → タブタイトルがフォルダ名に更新される
4. 画像が表示される → タイトルが「フォルダ名 | 画像名」に更新される
5. `×` クリック → タブが閉じる

## エラーケース

| 条件 | 挙動 |
|------|------|
| 通常タブが1枚の時に `×` クリック | 何もしない（閉じない）|
| お気に入りタブが既に開いている時に `☆` クリック | お気に入りタブにフォーカスを移動 |

## 未対応ケース

- タブの並び替え後の順序保存（movable=True だが保存されない）
- キーボードショートカット
