# image-viewer 概要

## 目的・背景

Windows 上で動作する画像ビューアーアプリ。フォルダ単位で画像を管理・閲覧することを主目的とし、マウス操作のみで快適に画像を切り替えられる UI を提供する。

## スコープ

### 作るもの

- フォルダ選択ダイアログ（起動時または任意タイミング）
- 左パネル：選択フォルダを起点としたサブフォルダツリー（折りたたみ可）
- 右パネル：選択フォルダの画像ビューアー
  - フィット表示（デフォルト）
  - マウスホイールによるズームイン・アウト
  - マウスサイドボタン（進む/戻る）による画像切り替え
  - 下部ホバーで表示されるスライダー（複数画像時）
- 隣接画像のプリロードによる高速切り替え（100ms 以内）

### 作らないもの

- 画像編集・加工機能
- スライドショー・自動再生
- EXIF・メタデータ表示
- Windows 以外の OS 対応

## 制約

- Python 3.10 以上、PyQt6 を使用
- Windows 10/11 での動作を前提
- フォルダ切り替え時の初期読み込みは数百ミリ秒まで許容
- フォルダ内の画像切り替えは 100ms 以内を目標

## 完了条件

- フォルダを選択してサブフォルダツリーが表示される
- ツリーのフォルダ選択で画像ビューアーに先頭画像が表示される
- マウスサイドボタンで前後の画像に切り替えられる（循環）
- マウスホイールで画像をズームイン・アウトできる
- 複数画像のあるフォルダで下部ホバー時にスライダーが表示される
- スライダーで任意の画像に切り替えられる（右=先頭、左=末尾）
- 前後切り替え・スライダー操作が 100ms 以内で完了する
