# enhancements 概要

## 目的・背景

初期実装後にユーザーから追加された操作性の改善要望をまとめた機能グループ。独立して実装・テストできる項目が複数あるため、サブ項目としてまとめた。

## スコープ

### 作るもの

- ズーム中の画像ドラッグパン（移動）、端を超えないクランプ
- ホバースライダーのクリックジャンプ（QSlider のページステップを廃止）
- 兄弟フォルダをまたいだ前後画像切り替え
- フォルダ右クリックコンテキストメニュー（エクスプローラーで開く・お気に入りに追加・別タブで開く）
- フォルダビューの `<`/`>` 折りたたみトグルボタン
- マウスホイール↔サイドボタン操作入れ替えスワップボタン（右上オーバーレイ）

### 作らないもの

- キーボードショートカット
- タッチ操作対応
- タブごとの個別スワップ設定（グローバル設定として扱う）

## 制約

- 既存コンポーネントの構造を大きく変えない
- スワップ設定は状態保存で永続化する

## 完了条件

- ズーム中のドラッグで画像が移動し、画像端で止まる
- スライダークリックで該当画像にジャンプする
- 先頭/末尾で前後操作をすると兄弟フォルダへ移動する
- コンテキストメニューの全項目が動作する
- トグルボタンでフォルダビューが折りたたみ/展開でき、状態に応じて `<`/`>` が変わる
- スワップボタンでホイールとサイドボタンの動作が入れ替わり、設定が保存される
