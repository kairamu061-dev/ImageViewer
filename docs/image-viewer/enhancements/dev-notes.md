# enhancements 開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| トグルボタンをスプリッター外に配置 | スプリッターが幅0になってもボタンは常時表示される必要があるため |
| `setSizes()` 後に `_update_toggle_btn` を明示的に呼ぶ | `setSizes()` は `splitterMoved` シグナルを発火しないため、ボタンテキストが更新されなかった |
| スワップ設定をグローバルにする | タブごとに異なる設定は管理が複雑で、ユーザーにとっても混乱しやすいため |
| `_zoom_pixmap` キャッシュを追加 | ドラッグ中はスケールが変わらないため、毎フレームの `pixmap.scaled()` をゼロコストにできる |

## 発生した問題と対処

| 問題 | 対処 |
|------|------|
| ホイール移動モードで上スクロールが next になっていた | `delta > 0` = 前の画像に修正 |
| フォルダパネル折りたたみ後もボタンが `<` のまま | `_toggle_folder_panel` 内で `_update_toggle_btn` を明示呼び出しに修正 |
| ズーム中に画像が画面外まで飛ぶ | `_clamp_offset` を追加しドラッグ・ズーム後に適用 |
