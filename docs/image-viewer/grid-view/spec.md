# grid-view 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | モード切替ボタン | ビューアー左上のオーバーレイボタン。クリックで 1枚表示 ⇔ 一覧表示を切り替え |
| 2 | サムネイルグリッド | 横5枚固定・正方形セル。縦スクロールでフォルダ内全画像を表示 |
| 3 | 非同期サムネイル読み込み | バックグラウンドで縮小デコードし、順次表示。読み込み中はプレースホルダ |
| 4 | サムネイルから1枚表示 | サムネイルをクリックするとその画像の1枚表示に切り替え |
| 5 | 現在位置の同期 | 一覧表示に入ると現在の画像が選択状態になり、その位置までスクロール |

## 画面・操作フロー

1. 通常は従来どおりの1枚表示（左上に「一覧」ボタン）
2. 「一覧」クリック → グリッド表示に切替。現在の画像が選択され、見える位置までスクロール
3. ホイール / スクロールバー → 縦スクロールでフォルダ内全画像を閲覧
4. サムネイルクリック → その画像の1枚表示に切替（ボタンは「一覧」に戻る）
5. グリッド表示中に「1枚」クリック → 元の画像のまま1枚表示に戻る
6. フォルダを切り替え → 表示中のモードを維持したまま新フォルダの内容を表示

## 各画面 / 状態の詳細

### 1枚表示（既存）

- **表示内容**: 従来の ImageCanvas。左上「一覧」、右上「W:移動 / W:拡縮」
- **操作**: 従来どおり（ホイール移動・拡縮、下端ホバースライダー）
- **遷移先**: 「一覧」クリックで一覧表示へ

### 一覧表示

- **表示内容**: 5列のサムネイルグリッド。セルはビューアー幅/5 の正方形。ファイル名はツールチップで表示。読み込み中のセルは暗色プレースホルダ
- **操作**: 縦スクロール（ホイール1ノッチ=2/5行、スクロールバー矢印も同幅）、サムネイルクリック
- **遷移先**: サムネイルクリック or 「1枚」クリックで1枚表示へ
- スワップボタン（W:移動/拡縮）は1枚表示専用のため一覧表示中は非表示
- 下端ホバーのジャンプスライダーも一覧表示中は無効

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 画像0枚のフォルダで一覧表示 | 空のグリッド（何も表示しない） |
| 読み込めない画像ファイル | プレースホルダのまま表示（クリックは可能） |
| 一覧表示中にウィンドウリサイズ | セルサイズを再計算し常に横5枚を維持 |

## 未対応ケース

- 列数のカスタマイズ（5列固定）
- サムネイルの右クリックメニュー・ファイル操作
- 表示モードの永続化（再起動後は1枚表示に戻る）
