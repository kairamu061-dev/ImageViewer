# grid-view 概要

## 目的・背景

現在のビューアーは1枚ずつの表示のみで、フォルダ内にどんな画像があるかを一覧で把握できない。
サムネイルのグリッド表示（分割表示）モードを追加し、フォルダ内の全画像を一覧・スクロールで確認できるようにする。

## スコープ

### 作るもの

- ビューアー左上のモード切替ボタン（1枚表示 ⇔ 一覧表示）
- 一覧表示モード: 横5枚固定のサムネイルグリッド、縦スクロールでフォルダ内全画像を表示
- サムネイルの非同期読み込み（UI をブロックしない）
- サムネイルクリックでその画像の1枚表示に切り替え
- 一覧表示に入った時、現在の画像位置までスクロール・選択表示

### 作らないもの

- 列数の変更 UI（5列固定）
- サムネイルからのファイル操作（削除・リネーム・D&D など）
- サブフォルダ画像の合算表示

## 制約

- 既存の ImageViewerPanel / ImageCache の構造を大きく変えない
- サムネイルはデコード時に縮小し、原寸画像をメモリに保持しない
- 通常タブ・お気に入りタブの両方で動作すること（ImageViewerPanel 内で完結させる）

## 完了条件

- 左上のボタンで 1枚表示 ⇔ 一覧表示を切り替えられる
- 一覧表示で横に5枚並び、縦スクロールでフォルダ内の全画像が見える
- サムネイルクリックでその画像が1枚表示される
- 大量画像フォルダでも UI が固まらない
