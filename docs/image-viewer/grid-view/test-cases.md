# grid-view テストケース

## ユニットテスト

| ID | テスト名 | 入力 / 条件 | 期待結果 | ステータス |
|----|---------|------------|---------|-----------|
| U-01 | 縮小デコード | 長辺 256px 超の画像 | 長辺 256px 以下・アスペクト比維持で QImage が返る | [ ] |
| U-02 | 世代破棄 | set_images 直後に再度 set_images | 旧世代の読込結果がグリッドに反映されない | [ ] |

## インテグレーションテスト

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| I-01 | モード切替 | 画像のあるフォルダを表示中 | 左上「一覧」クリック → 「1枚」クリック | グリッド表示 ⇔ 1枚表示が切り替わる。スワップボタンは一覧中のみ非表示 | [ ] |
| I-02 | 現在位置同期 | 10枚目を1枚表示中 | 「一覧」クリック | 10枚目が選択状態で見える位置にスクロールされる | [ ] |
| I-03 | サムネイルから遷移 | 一覧表示中 | 任意のサムネイルをクリック | その画像の1枚表示に切り替わり、ボタンが「一覧」に戻る | [ ] |
| I-04 | フォルダ切替 | 一覧表示中 | フォルダツリーで別フォルダを選択 | 一覧表示のまま新フォルダのサムネイルが表示される | [ ] |

## E2Eテスト

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 5列維持 | 一覧表示中 | ウィンドウ幅を大きく変更・フォルダパネルを開閉 | 常に横5枚で並び、横スクロールバーが出ない | [ ] |
| E-02 | 全画像スクロール | 50枚以上のフォルダで一覧表示 | 最下部まで縦スクロール | 全画像のサムネイルが表示される | [ ] |
| E-03 | 大量画像 | 1000枚規模のフォルダ | 一覧表示に切替え操作を続ける | UI が固まらず、サムネイルが順次埋まる | [ ] |
| E-04 | 空フォルダ | 画像0枚のフォルダ | 「一覧」クリック | 空のグリッドが表示されエラーにならない | [ ] |
| E-05 | 破損ファイル | 読み込めない画像を含むフォルダ | 一覧表示 | 該当セルはプレースホルダのまま。クリックすると1枚表示(「画像がありません」相当) | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
