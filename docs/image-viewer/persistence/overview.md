# persistence 概要

## 目的・背景

アプリを閉じて再度開いたときに前回の状態を復元する。タブ構成・選択フォルダ・画像インデックス・お気に入い・スワップ設定・ウィンドウ位置を保存対象とする。

## スコープ

### 作るもの

- アプリ終了時に状態を JSON ファイルへ保存
- 起動時に JSON ファイルから状態を復元
- 保存対象：タブ一覧・各タブの状態・お気に入いデータ・スワップモード・ウィンドウ位置とサイズ

### 作らないもの

- クラウドへの同期
- 複数プロファイルの管理
- 手動でのエクスポート/インポート

## 制約

- 保存先は `~/.image_viewer_state.json`（ユーザーホームディレクトリ）
- 保存が失敗しても例外を出さずサイレントに失敗
- 復元時に存在しないパスはスキップ

## 完了条件

- アプリを閉じて再起動すると同じタブ構成で開く
- 各タブの選択フォルダと表示画像が復元される
- お気に入いの内容が復元される
- スワップモード設定が復元される
- ウィンドウ位置・サイズが復元される
