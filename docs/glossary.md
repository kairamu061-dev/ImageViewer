# 用語集

このプロジェクトで使用するドメイン固有の用語を定義する。
実装・設計・仕様書内で用語を使う際はここに定義されたものに統一する。

---

| 用語 | 定義 |
|------|------|
| フォルダビュー | アプリ左パネル。ルートフォルダを起点としたサブフォルダのみのツリー表示 |
| 画像ビューアー | アプリ右パネル。選択フォルダの画像を表示するエリア |
| フィット表示 | 画像をビューアーサイズに縦横比を保ちつつ収める表示モード（デフォルト）|
| ズームモード | マウスホイールまたはサイドボタンで拡縮した後の表示モード |
| ホバースライダー | ビューアー下部にマウスを近づけたときに表示される画像切り替えスライダー |
| スワップモード | ホイールとサイドボタンの操作を入れ替えるトグル設定 |
| プリロード | 現在表示中の画像の前後 N 枚をバックグラウンドで先読みする処理 |
| お気に入り | ユーザーが登録したフォルダを管理する専用タブ |
| 階層 | お気に入り内でフォルダを整理するための仮想グループ（実フォルダは作られない）|
| ルートフォルダ | フォルダビューの起点となるフォルダ。「フォルダを開く」で選択する |
| 兄弟フォルダ | 現在選択中のフォルダと同じ親フォルダを持つ別のフォルダ |
| 世代番号 | キャッシュの無効化に使うカウンター。フォルダ切り替え時にインクリメントされる |
| タブコンテンツ | 1つのタブが持つフォルダビュー＋画像ビューアーのセット |
